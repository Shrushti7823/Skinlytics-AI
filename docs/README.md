## Overview

Skinlytix AI is an AI-powered skincare platform that helps users analyze skin concerns and receive personalized skincare recommendations.

The application combines Machine Learning models with a modern web interface to provide:

- 🧴 Skin analysis
- 🤖 AI-based skincare recommendations
- 💊 Personalized skincare routines
- 📈 Product recommendations
- 👤 User profile management
- 📜 Analysis history
- 🌱 Skin type-based filtering
